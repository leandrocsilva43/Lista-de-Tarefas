# Lista de Tarefas ✅

Aplicação de gerenciamento de tarefas com armazenamento local via `localStorage`. As tarefas persistem mesmo após fechar o navegador.

---

## 🚀 Funcionalidades

- Adicionar e remover tarefas
- Marcar tarefas como concluídas
- Dados salvos no `localStorage` — sem necessidade de backend

---

## 🛠️ Tecnologias utilizadas

- **JavaScript** — lógica e manipulação do DOM
- **HTML5** — estrutura
- **CSS3** — estilização

---

## ▶️ Como rodar

Clone o repositório e abra o `index.html` diretamente no navegador.

```bash
git clone https://github.com/leandrocsilva43/Lista-de-Tarefas.git
```
